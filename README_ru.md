Данный репозиторий содержит исходный код и методологию для исследовательского проекта **«Статистический анализ устойчивости методов паншарпенинга для классификации ландшафтов»**.

Исследование выполняет комплексный сравнительный анализ различных алгоритмов паншарпенинга, оценивая их спектральную точность, пространственное качество и устойчивость в различных типах территории (городские, лесные, сельскохозяйственные) с использованием статистического подхода. Конечная цель — предоставить основанные на данных рекомендации по выбору оптимальных методов паншарпенинга для повышения точности классификации спутниковых изображений.

## 📖 Обзор

Спутники среднего разрешения, такие как Landsat 8, предоставляют ценные данные, но их пространственное разрешение ограничивает возможности детального анализа ландшафтов. Методы паншарпенинга направлены на повышение этого разрешения путем слияния мультиспектральных (MS) каналов с высокодетальным панхроматическим (PAN) каналом. Однако, производительность и стабильность этих методов могут значительно различаться в зависимости от типа ландшафта.

Данное исследование решает эту проблему путем:
*   Реализации широкого спектра классических и современных алгоритмов паншарпенинга.
*   Оценки их производительности с использованием безэталонных метрик качества.
*   Применения устойчивой статистической методологии (консенсусное ранжирование, bootstrap-анализ, анализ согласованности) для оценки стабильности результатов.
*   Анализа влияния паншарпенинга на точность контролируемой классификации.

## 🚀 Основные возможности

*   **Сквозной пайплайн:** Полный, автоматизированный пайплайн от предобработки данных до статистического анализа и визуализации.
*   **Множество алгоритмов паншарпенинга:** Реализация 9 различных методов из категорий Подстановки Компонент (CS), Мультиразрешающего Анализа (MRA) и Гибридных методов (Hybrid/Model-based).
*   **Безэталонная оценка качества:** Оценка с использованием метрик QNR, Gradient Similarity, Entropy и Spatial Correlation.
*   **Устойчивый статистический анализ:** консенсусное ранжирование, bootstrap-анализ и анализ согласованности на основе коэффициента Кендалла W.
*   **Анализ по биомам:** Оценка производительности и стабильности методов для городских, лесных и сельскохозяйственных ландшафтов.
*   **Контролируемая классификация:** использование алгоритмов Maximum Likelihood, Random Forest и XGBoost для оценки практического эффекта паншарпенинга.


## 🛠️ Реализованные методы

### Алгоритмы паншарпенинга
*   **Подстановка Компонент (CS):**
    *   BT (Brovey Transform)
    *   BT-H (Brovey с согласованием гистограмм)
    *   PCA (Principal Component Analysis)
*   **Мультиразрешающий Анализ (MRA):**
    *   ATWT (À Trous Wavelet Transform)
    *   HPF (High-Pass Filtering)
    *   SFIM (Smoothing Filter-based Intensity Modulation)
*   **Гибридные/Модельные методы (Hybrid/Model-based):**
    *   GS (Gram-Schmidt)
    *   GS2 (Gram-Schmidt Adaptive)
    *   PRACS (Partial Replacement Adaptive Component Substitution)

### Алгоритмы классификации
*   Метод максимального правдоподобия (Maximum Likelihood)
*   Случайный лес (Random Forest)
*   XGBoost

## ⚙️ Установка

Для установки и запуска проекта локально выполните следующие шаги.

### 1. Клонирование репозитория

```bash
git clone https://github.com/TovarishchF/pansharpening-robustness-analysis.git
cd pansharpening-robustness-analysis
```
### 2. (Рекомендуется) Создание виртуального окружения
```bash
python -m venv .venv
```
Активация окружения:

* Git Bash / Linux / macOS
```bash
source .venv/bin/activate
```
* Windows (PowerShell / CMD)
```bat
.venv\Scripts\activate
```
### 3. Установка зависимостей
```bash
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

## 🔁 Воспроизводимость

Все эксперименты в рамках данного проекта полностью воспроизводимы с использованием предоставленного исходного кода, конфигурационных файлов и фиксированных значений генераторов случайных чисел, указанных в config.yaml.

и следующей информации о сцене:

Идентификатор продукта Landsat: LC08_L1TP_185018_20250718_20250726_02_T1

Дата съёмки: 18 июля 2025 г.

Спутник / сенсор: Landsat 8 OLI/TIRS

Путь / ряд (WRS-2): 185 / 18

Все данные Landsat, использованные в исследовании, находятся в открытом доступе и распространяются Геологической службой США (USGS) без ограничений на использование.
